myAdvice Student Advising System
COMP.2800 - D.3 Project Code

Deployment & Running Documentation
1. Clone or download this repository.
2. Navigate to the `out/production/myAdviceSystem` directory to locate the compiled `.class` executables.
3. Run the application from the terminal using: `java Main` (or run it directly through an IDE like IntelliJ by executing `Main.java`).

Database Connection Instructions
Please note: For this deliverable, the application utilizes a static, in-memory `DataStore` to simulate database interactions. There is no external MariaDB/SQL database required to be installed or connected to run this pilot application.
